# To-Do List Manager

## Overview
A productivity tool that allows users to create, manage, and track daily tasks.

The program stores tasks and allows users to update their progress.

## Difficulty

Beginner

## Core Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Save tasks to a file
- View task list

## Example Usage
```
$ python todo.py add "Finish homework"
Task added.
```

## Concepts Used

- Lists
- File persistence
- Functions
- CLI menu systems

## Possible Extensions

- Task priorities
- Due dates
- Calendar integration
