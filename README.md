## Password Manager 

### Overview
A command-line password manager that securely stores and retrieves user passwords for different accounts.

This project focuses on security fundamentals, file storage, and user authentication.

### Difficulty

Intermediate

### Core Features

- Master password authentication
- Store account credentials
- Generate strong passwords
- Search saved accounts
- Encrypt stored data

### Example Usage
```
$ python manager.py add
Account: github
Username: johndoe
Password saved successfully.
```

### Concepts Used

- File I/O
- Encryption / hashing
- Dictionaries
- Command-line arguments
- Error handling

### Possible Extensions

- Clipboard copy feature
- GUI version
- Cloud synchronization

## To-Do List Manager

### Overview
A productivity tool that allows users to create, manage, and track daily tasks.

The program stores tasks and allows users to update their progress.

### Difficulty

Beginner

### Core Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Save tasks to a file
- View task list

### Example Usage
```
$ python todo.py add "Finish homework"
Task added.
```

### Concepts Used

- Lists
- File persistence
- Functions
- CLI menu systems

### Possible Extensions

- Task priorities
- Due dates
- Calendar integration
