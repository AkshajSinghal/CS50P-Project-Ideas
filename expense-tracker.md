# Expense Tracker

## Overview
A program that helps users track their daily expenses and analyze spending habits.

## Difficulty

Intermediate

## Core Features

- Add income and expenses
- Categorize transactions
- Calculate totals
- Monthly summaries
- Export data

## Concepts Used

- Dictionaries
- CSV or JSON storage
- Data aggregation
- Basic statistics

## Possible Extensions

- Visual charts
- Budget limits
- Mobile version
